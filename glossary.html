Glossary of ICT Terms

Key Concepts and Tools in Collaboration Technology

Accessibility

Designing tools and platforms so that people with disabilities can use them effectively.

Brainstorming Tools

Applications that help a group generate ideas and collaborate creatively, e.g., Miro or Freehand.

Cloud Collaboration

Co-authoring and editing documents online in real time using cloud platforms like Google Docs or OneDrive.

Co-editing

Multiple users working simultaneously on the same document or file with updates reflected in real-time.

Digital Whiteboard

A virtual space for collaborative drawing, writing, and sharing ideas in real time.

Encryption

The process of encoding data to prevent unauthorized access or tampering.

File Sharing

Distributing digital files among users via email, cloud storage, or collaboration platforms.

Hybrid Work

A work model combining remote and in-office work for flexibility and efficiency.

Kanban Board

A visual project management tool that shows tasks in columns (e.g., To Do, In Progress, Done) to manage workflow.

Network Security

Protecting a computer network and its data from unauthorized access, viruses, or attacks.

Online Meeting

A virtual meeting conducted via video or audio over the internet using tools like Zoom or Google Meet.

Project Management Software

Applications that help teams organize, track, and manage tasks, deadlines, and workflows, e.g., Trello, Asana, Jira.

Remote Access

The ability to access a computer, network, or files from a remote location, often using secure connections.

Screen Sharing

Broadcasting your computer screen to other users in real-time for presentations, tutorials, or collaboration.

Task Management

Features or tools that allow assigning, tracking, and completing tasks within a team or project.

Unified Communications (UC)

Integrating multiple communication methods—voice, video, messaging—into a single platform.

Video Conferencing

Conducting meetings or discussions via video and audio over the internet.

Wiki

A collaborative website that allows users to create, edit, and organize content together in real time.

Zoom

A popular platform for video conferencing, webinars, and virtual meetings.

Slide Master

The base layout template in PowerPoint used to maintain consistency across slides.
