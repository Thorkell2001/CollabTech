CollabTech

Web-Based Collaboration Tool Study Guide

Welcome

This website is a student-made study guide developed for the research titled:

“CollabTech: Web-Based Guide for Using Collaboration Tools Among ICT Students in Antipolo City Senior High School.”

This site is intended for learning and familiarization only. All pre-tests and post-tests for the study are administered using printed test papers.

Collaboration Tool Study Guide

Microsoft Word

  • Use Track Changes for group editing
  • Add and reply to comments
  • Share documents via OneDrive
  • Review and restore version history

Microsoft PowerPoint

  • Edit slides collaboratively in real time
  • Use comments for feedback
  • Share presentation links safely
  • Access version history

Microsoft Excel

  • Share workbooks for team tasks
  • Use comments and notes
  • Apply basic formulas in group work
  • Monitor changes in shared files

Canva

  • Invite team members to a design
  • Assign editor or viewer roles
  • Use comments and suggestions
  • Collaborate in real time

About the Research

Purpose: To determine whether a student-made web-based guide can improve the collaboration tool proficiency of Grade 12 ICT students.

Design: Experimental research using control and experimental groups with paper-based pre-test and post-test.

Note: This website does not collect personal data, test results, or user information.
