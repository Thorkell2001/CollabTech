Common Problems & Fixes

Quick Solutions for ICT Tools

Problem 1: File Didn’t Save

Cause: Program closed before saving, or wrong folder.

Fix: Check Recent Documents, AutoRecover, Ctrl+S frequently, save in familiar folders.

Problem 2: Formatting Looks Different

Cause: Fonts not installed on other device.

Fix: Save as PDF, Embed Fonts in Options → Save.

Problem 3: Slow Canva Designs

Cause: Too many elements, large files, internet issues.

Fix: Close tabs, compress images, use Canva desktop app, refresh regularly.

Problem 4: Images Move in Word/PowerPoint

Cause: Improper text wrapping.

Fix: Word → Right-click → Wrap Text → In Line or Square.
PowerPoint → Align → Lock Object.

Problem 5: Excel Formulas Not Working

Cause: Typing errors or wrong references.

Fix: Start with =, check spaces, use Show Formulas.

Problem 6: Canva Fonts/Templates Locked

Cause: Pro-only resources.

Fix: Use free alternatives, upload own fonts if allowed, stick to free elements.

Problem 7: Presentation Doesn’t Play

Cause: Media not embedded.

Fix: Insert → Media → Embed, save as .pptx, test before class.

Problem 8: Computer Freezes

Cause: Too many apps running.

Fix: Ctrl+Shift+Esc → End unresponsive tasks, restart, clean up files.
