Microsoft Word

Word Processing for Students

Overview

Microsoft Word is used to create, edit, and format text documents such as essays, research papers, reports, and newsletters. It helps students present work professionally.

Basic Features

  • Text Formatting: Bold, Italic, Underline, Font styles and sizes
  • Paragraph Formatting: Alignment, Spacing, Indentation
  • Insert Tools: Pictures, Tables, Shapes, Charts
  • References: Page numbers, headers/footers, citations
  • Review: Track Changes, Add Comments

Step-by-Step: Create a Research Paper

  1. Open Word → Blank Document
  2. Layout → Margins → Normal (1 inch)
  3. Font → Times New Roman, Size 12
  4. Line Spacing → Double
  5. Add Title Page → Insert → Cover Page
  6. Insert Page Numbers → Top of Page
  7. Add References → References Tab → Insert Citation
  8. Save as .docx → Export as PDF

Tips & Shortcuts

  • Ctrl + B/I/U → Bold/Italic/Underline
  • Ctrl + C/V → Copy/Paste
  • Ctrl + Z → Undo
  • Ctrl + F → Search text
  • Use Styles → Quickly format headings and subtitles

Common Mistakes & Fixes

  • Document looks different → Save as PDF
  • Deleted text → Ctrl + Z immediately

Mini Activity

Create a 1-page assignment with:

  • Title in Bold
  • Subtitle in Italics
  • Paragraph (5–7 sentences)
  • Image inserted → format “Square”
  • Page numbers → bottom center
