Overview

CollabTech is a student-made instructional website developed as part of a research study entitled “CollabTech: Web-Based Guide for Using Collaboration Tools Among ICT Students in Antipolo City Senior High School.”

This website serves as a supplementary learning material designed to help Grade 12 ICT students understand, explore, and properly use productivity and collaboration tools commonly required in academic tasks and group projects.

Purpose of the Guide

The primary purpose of this web-based guide is to improve students’ understanding of collaboration tools by providing clear explanations, practical examples, and conceptual discussions rather than simple memorization.

The guide supports independent learning and helps students build confidence in using digital tools for academic collaboration.

How to Use This Website

Students are encouraged to read each module carefully and reflect on how the tools are applied in real academic situations. This guide is designed to complement classroom instruction and printed assessments.

The website does not include online tests. All assessments related to this guide are administered through printed pre-test and post-test instruments as part of the research design.
