CollabTech

A Student-Made Web-Based Guide for Using Collaboration Tools

CollabTech is a student-made instructional website developed as part of a research study entitled “CollabTech: Web-Based Guide for Using Collaboration Tools among Grade 12 ICT Students of Antipolo City Senior High School.”

Overview

This website serves as a supplementary learning material designed to help Grade 12 ICT students understand, explore, and properly use productivity and collaboration tools commonly required in academic tasks and group projects.

Purpose of the Guide

The primary purpose of this web-based guide is to improve students’ understanding of collaboration and productivity tools by providing clear explanations, practical examples, and conceptual discussions rather than simple memorization.

This guide supports independent learning and helps students build confidence in using digital tools for academic collaboration.

How to Use This Website

Students are encouraged to read each module carefully and reflect on how the tools are applied in real academic situations. This website is designed to complement classroom instruction and printed assessments.

This website does not include online tests. All assessments related to this guide are administered through printed test instruments as part of the research design.
