Microsoft Excel

Data Management and Analysis

What is Excel?

Excel is a spreadsheet program used to store, organize, calculate, and analyze data. Students can use it for budgets, charts, lists, and reports.

Basic Components

  • Workbook: The whole Excel file.
  • Worksheet: A single sheet (tab) inside the workbook.
  • Ribbon: Toolbar at the top with Home, Insert, Formulas, etc.
  • Cell: Intersection of a row and column where data is typed.
  • Rows & Columns: Rows are horizontal numbers, columns are vertical letters.
  • Formula Bar: Shows or edits the content of a selected cell.

Basic Formulas

  • Add: =A1+A2
  • Subtract: =A1-A2
  • Multiply: =A1*A2
  • Divide: =A2/A1
  • Sum: =SUM(A1:A10)
  • Average: =AVERAGE(A1:A10)
  • Max/Min: =MAX(A1:A10) / =MIN(A1:A10)

Quick Formatting Tips

  • Bold/Italic/Underline: Ctrl+B / Ctrl+I / Ctrl+U
  • Change Fill Color: Alt + H + H
  • Copy Formatting: Use Format Painter (paintbrush icon)
  • Insert Date/Time: Ctrl+; or Ctrl+Shift+:
  • Freeze Panes: View → Freeze Panes → Keep headers visible while scrolling
  • Sorting & Filtering: Data → Sort & Filter → Arrange or Show selected rows
  • Autofill: Drag the small square at the corner of a cell to fill series automatically

FAQs

  • Q: What is Excel used for?
    A: Storing, organizing, calculating, and analyzing data.
  • Q: Difference between Workbook & Worksheet?
    A: Workbook = whole file, Worksheet = single sheet.
  • Q: What is a PivotTable?
    A: Summarizes and analyzes large amounts of data quickly.
  • Q: How do I create a chart?
    A: Select data → Insert → Choose chart type.
